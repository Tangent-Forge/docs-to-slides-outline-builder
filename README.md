# Docs→Slides Outline Builder

**Version:** 1.0.0  
**Status:** Development  
**Target Launch:** Q2 2026  

## Overview

Docs→Slides Outline Builder is a Google Workspace add-on that converts Google Docs content into structured Google Slides outlines. It helps users create presentation outlines from documentation quickly and efficiently.

## Key Features

### Smart Conversion

- **Structure Extraction**: Identifies headings and creates slide structure
- **Content Mapping**: Maps Docs content to Slides text
- **Auto-Formatting**: Applies consistent slide formatting
- **Template Support**: Uses customizable slide templates

### Customization

- **Slide Templates**: Choose from pre-built templates
- **Content Filters**: Select which content to include
- **Preview Mode**: Preview outline before creating slides
- **Batch Processing**: Convert multiple Docs at once

### Integration

- **Docs API**: Reads Docs content
- **Slides API**: Creates and formats Slides
- **Direct Creation**: Creates new presentations automatically
