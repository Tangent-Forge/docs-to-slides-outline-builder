- **Data Access**: The add-on reads Google Docs content (headings, text) for outline generation. No file content is stored permanently.
- **Data Storage**: No data is stored permanently. Presentations are created in Google Drive.
- **Data Sharing**: No data is sent to or shared with third parties. All processing happens within Google's infrastructure.
- **File Operations**: The add-on creates new Google Slides presentations based on Docs content.
- **Logging**: Operational logs are written to Apps Script Logger for troubleshooting.
- **Removal**: Remove stored settings by uninstalling the add-on or clearing Script Properties.
- **Support**: For privacy questions, open an issue in the repository or contact support@tangentforge.com.
